# yatranslate
Script for auto-translate through yandex api.  
For working of an application, you have to get a [token](https://translate.yandex.ru/developers/keys) Yandex API.  

## Depencies:
`yum install notification-daemon xsel`
## Build 
```bash
make build
```
## Install
```bash
make install
```
## Uninstall

```bash
make uninstall
```
## Support languages:
en -> ru
